# PO Review Board

Review .po translations with diff view, comments, and approval workflow.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install po-review-board
```
